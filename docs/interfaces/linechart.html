Optional area

area: Area
  • Defined in config.spec.d.ts:509

Configures the chart to display a shaded area instead of (or in addition to) a line. This can be useful for showing a range or interval, like a statistical uncertainty interval.

Optional cross_schema

cross_schema: CrossSchema
  • Defined in config.spec.d.ts:518

Display data from a schema other than the currently selected schema.

The schema indicated must include the dimensions referenced as domain and split_dimension, if applicable. If this field is not supplied, the current schema will be queried.

Optional split_dimension

split_dimension: undefined | string
  • Defined in config.spec.d.ts:503

Dimension whose options will be rendered as separate lines (and/or areas); references Dimension.name. If not provided, only a single line/area will be rendered.
