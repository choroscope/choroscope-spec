Interface CustomLegend

Custom SVG (Scalable Vector Graphics) element to be used in place of the usual legend

Note that the (outermost) SVG element must define the display dimensions. The simplest way to do that is by defining the height and width attributes in terms of pixels, e.g.:

<svg height="100" width="100" ...>...</svg>
