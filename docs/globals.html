InfoDisplay: BarChart | LineChart | ValuesDisplay
  • Defined in config.spec.d.ts:550

Configuration for an info display component

In addition to visualizing data on the map, the application also provides an "Info" box for showing supplemental data about the selected pixel (in "geospatial" mode) or geographic feature (in "aggregate" mode). This Info Box can render one or more display components, defined by the configuration object(s) provided.
